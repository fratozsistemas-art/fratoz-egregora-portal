# Fratoz Egrégora

Rede colaborativa de arte e cultura que conecta oito linguagens artísticas em experiências transmídia imersivas.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Desenvolvimento

```sh
npm install
npm run dev
```

## Sobre

Fratoz Egrégora é um portal de arte e cultura que reúne Cinema, Teatro, Música, Fotografia, Pintura, Dança, Literatura e Escultura em uma experiência digital imersiva e transmidiática.
